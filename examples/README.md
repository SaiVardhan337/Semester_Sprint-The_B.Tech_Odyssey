# Usage & Examples 💡

This directory contains standalone, isolated mini-demos demonstrating core modular components of **Semester Sprint: The B.Tech Odyssey**.

---

## 📁 Included Examples

### 1. `standalone-audio-synth.html`
* **Description**: A self-contained HTML page demonstrating the programmatic Web Audio API chiptune synthesizer engine used in the game.
* **Features**: Live playable buttons for chiptune background music, jump sound effect, collection jingle, and hit sound effects using pure Web Audio API synthesis (zero external audio files).
* **How to run**: Double-click `standalone-audio-synth.html` to open in any web browser.

### 2. `canvas-runner-basic.html`
* **Description**: A lightweight 80-line HTML5 Canvas side-scroller prototype.
* **Features**: Demonstrates basic player physics (gravity, jumping, sliding, ground collision) and scrolling ground renderer in clean, unminified JavaScript.
* **How to run**: Double-click `canvas-runner-basic.html` to open in any web browser.
